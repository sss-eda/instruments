## Settings
A list of things that can be configured:

    - OpenAPI (HTTP) (JSON)

    - GRPC (GRPC) (PROTOBUF)
    - NATS (NATS) (PROTOBUF or JSON)
* Repostiory Types:
    - Postgres
    - Memory
